Positioning Techniques Demo

Static Positioning

This is static positioning. Elements are in normal document flow.

Relative Positioning

This is relative positioning. Elements are positioned relative to their normal position.

Relative Child

Absolute Positioning

This is absolute positioning. Elements are positioned relative to the nearest positioned ancestor.

Absolute Child

Fixed Positioning

This is fixed positioning. Elements are positioned relative to the viewport.

Sticky Positioning

This is sticky positioning. Elements are initially in normal flow, but become fixed based on scroll position.
